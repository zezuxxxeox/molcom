# AI Privacy Guard

웹캠에 본인 외 인물이 0.2초 이상 연속 감지되면 보호 화면을 띄우거나 지정한 창을 앞으로 가져오는 Windows용 Python MVP입니다.

## 실행

```powershell
python -m pip install -r requirements.txt
python privacy_guard.py
```

또는 `run.bat`을 실행하세요.

## 사용 순서

1. `얼굴 등록`을 눌러 본인 얼굴을 등록합니다.
2. 필요하면 설정을 조정합니다.
   - `위험 지속 시간`: 타인이 몇 초 이상 연속 감지되어야 작동할지 정합니다.
   - `쿨다운`: 한 번 작동한 뒤 다시 작동하지 않는 시간입니다.
   - `본인 판정 임계값`: 높을수록 엄격합니다. 오탐이 많으면 조금 낮추고, 타인을 본인으로 착각하면 높이세요.
   - `보호 방식`: `overlay`는 검은 전체 화면, `window`는 창 제목에 맞는 창을 앞으로 가져옵니다.
3. `감시 시작`을 누릅니다.

## 저장 데이터

얼굴 영상이나 사진은 저장하지 않습니다. 등록 시 추출한 얼굴 특징값만 `%USERPROFILE%\.ai_privacy_guard\owner_profile.json`에 저장합니다.

## 감지 로직

- 얼굴 없음: 아무 행동 없음
- 본인 얼굴만 있음: 안전
- 본인 외 얼굴 1명 이상: 위험 시간 누적
- 위험 시간이 기준 이상이고 쿨다운이 끝남: 보호 동작 실행

## 참고

이 MVP는 설치가 쉬운 MediaPipe 얼굴 랜드마크 기반 특징값을 사용합니다. 조명, 각도, 마스크, 안경, 카메라 품질에 따라 임계값 조정이나 재등록이 필요할 수 있습니다.
